¿Qué es prisma?

Prisma es un kit de herramientas de base de datos de código abierto. Se compone principalmente de las siguientes partes:

  • Prisma Client: generador de consultas autogenerado y seguro para Node.js y TypeScript
  • Prisma Migrate ( experimental ): sistema de migración y modelado de datos declarativos
  • Prisma Studio ( experimental ): GUI para ver y editar datos en su base de datos

Prisma Client se puede usar en cualquier aplicación de backend Node.js o TypeScript (incluidas las aplicaciones sin servidor y los microservicios). Puede ser una API REST, una API GraphQL, una API gRPC o cualquier otra cosa que necesite una base de datos.
